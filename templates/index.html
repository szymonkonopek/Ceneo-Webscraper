Libraries and sources used
BeautifulSoup : Python library used for pulling data out of HTML files.
Pandas : Open source data analysis and manipulation tool (Sortable table).
Flask : Light framework written in Python. Used for preparing HTML websites and calculations
Flask SQLAlchemy (SQLite) : Local database used for managing ceneo data.
Jinja2 Syntax : templating engine, used for transfering data between Python and HTML.
Sass : CSS with superpowers. Extension to CSS which helps in making websites more responsive.
